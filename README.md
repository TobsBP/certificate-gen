# 🎓 Certificate Gen

A web application built with **Angular** for generating and managing student certificates.

## ✨ Features

- 📋 List all generated certificates
- 🏅 Generate new certificates for students
- 🔍 View certificate details with generation date

## 🚀 Getting Started

### Prerequisites

- Node.js
- Angular CLI

### Installation

```bash
npm install
```

### Running locally

```bash
ng serve
```

Open your browser at `http://localhost:4200/`

## 🛠️ Built With

- [Angular 21](https://angular.dev/)
- [Vitest](https://vitest.dev/) — unit testing

## 🧪 Running Tests

```bash
ng test
```

## 📦 Build

```bash
ng build
```

Build artifacts will be stored in the `dist/` directory.
